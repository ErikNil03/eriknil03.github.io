About

Work

Summer Internship at Saab Technologies. During the summer of 2022, between June and August, I worked at Saab. In the course of my time there I worked together with other students on a group project. Together we programmed a Robot Watchdog which will patrol an area such as a hangar with random walking patterns. The dog will report any unauthorized persons detected and report to a central webapplication which acts as a control interface. The robotdog is a commercially available robot platform called Bittle made by Petoi LLC. If you're curious about what exactly we did, check out the project github here.

Projects

Senior High School Project. A study that explored how to digitally measure soil moisture via voltage with a microcomputer. Different soil types were studied and the correlations were shown in different regressions that can be used to specify the moisture, at a known voltage.

Java Payment Application. A java application made in Netbeans which is used for customers to order at restaurants. It comes with a graphical user interface, which can keep track of returning customers and print receipts. If the first and lastname matches with a former customer it will automatically fill in the rest of the order form as the customer ordered last time. Curious? See github.

Mario Game. Made as a High School Project in 2020 in Processing. It is a game made for fun. It has a main menu and a normal stage and after a while you can access the boss stage. Then you have to fight Bowser. If you manage to beat Bowser by throwing fireballs at him, you win! Then it takes you to another menu, which gives you the options to either back to main menu or play again. See github.
